# Cleaning Service Landing Page

Одностраничный сайт сервисной компании (клининг).

## Используемые технологии
- HTML5 (семантическая разметка)
- CSS3
- Flexbox
- Grid
- Адаптивная вёрстка (Desktop / Tablet / Mobile)

## Структура страницы
- Header с навигацией и выпадающим меню
- Hero-секция с основным оффером
- Блок преимуществ
- Блок процесса работы
- CTA-секция с формой
- Footer

## Адаптивность
- ≥1024px — desktop
- ≤1024px — tablet
- ≤425px — mobile

## Цель проекта
Практика семантической разметки, построения сеток (Flex/Grid)  
и адаптивной вёрстки без использования JavaScript.
